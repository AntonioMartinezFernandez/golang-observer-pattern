# golang-observer-pattern

Golang Observer Pattern Implementation in Golang

## Requisites

- Golang
- Air
- Docker

## Start

### Windows

```
air -c ./air.win.toml
```

### Linux / Mac

```
air -c ./air.dev.toml
```

### Docker

docker compose up
